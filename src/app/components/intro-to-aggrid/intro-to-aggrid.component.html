Intro to AG Grid

A first look at AG Grid in Angular — row data, column definitions, sorting, filtering, and cell renderers.

🎓 What you'll learn on this page
  • rowData Supply static arrays as grid data — each object maps to one row
  • ColDef Define columns with field, headerName, and display options
  • sortable Enable per-column sorting with a single boolean flag
  • filter Add text filters (filter: true) or typed filters (agNumberColumnFilter)
  • valueFormatter Transform raw values for display without changing the underlying data
  • cellRenderer Render custom content in a cell — e.g. booleans as emoji icons
  • defaultColDef Set shared column properties (width, style) that apply to every column
  • GridReadyEvent Load remote data via HttpClient once the grid is initialised
  • ColTypeDefs Define reusable column types and apply them with the type property
  • cellClass Apply CSS class names to cells for custom row-level styling
